## 프로젝트 소개

<p align="justify">
</p>
타학교에 비해 홍익대학교는 복잡한 건물 구조와 학교의 정보들이 흩어져있다. 헤이홍은 학교의 시설/관련 정보를 한눈에 모아볼 수 있는 커뮤니티형 어플이다.

</p>

## 구현 기능
<img src="./images/app_sub_img1.png"><img src="./images/app_sub_img2.png">
### 기능 1
시설 카테고리별 리스트 및 상세 조회
